# QuickShot

QuickShot is a Chrome Extension (Manifest V3) for ultra-fast area screenshots.

Primary flow:
1. Start capture.
2. Drag and select area.
3. QuickShot captures and crops the visible tab.
4. Cropped image is auto-copied to clipboard.
5. Editor opens for quick annotation and export (or can be skipped in Clipboard-only mode).

Current launch methods:
- Left click on extension icon (instant start, no popup).
- Keyboard shortcut `Ctrl+Shift+S`.
- Right-click context menu `Capture area with QuickShot`.

See [docs/README.md](docs/README.md) for detailed documentation.
