# Store Assets

## Required Chrome Web Store Screenshots
Capture these at either `1280x800` or `640x400`:

1. Start capture: dimmed page with selection rectangle visible.
2. Inline editor over the page after a capture.
3. Annotation controls open with color, width, and line settings visible.
4. A completed annotated screenshot showing arrows/shapes/text.
5. Options page with `Full` and `Clipboard only` modes visible.

## Recommended Store Copy Points
- Fast area screenshots from the current tab.
- Instant clipboard copy after capture.
- Lightweight inline editor for quick markup.
- Local-first: no backend, no analytics, no cloud upload.
