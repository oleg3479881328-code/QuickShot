# User Guide

## 1. Toolbar Usage
1. Left-click QuickShot icon in Chrome toolbar.
2. Drag to select an area.
3. Release mouse to finish capture.
4. Result is copied and editor opens (or is finished immediately in Clipboard-only mode).

## 2. Hotkey Usage
1. Press `Ctrl+Shift+S`.
2. Select area with mouse.
3. Use `Esc` to cancel if needed.

## 3. Context Menu Usage
1. Right-click on any page.
2. Choose `Capture area with QuickShot`.
3. Drag and release to capture.

## 4. Editing Workflow
1. QuickShot opens editor directly on the page in Full mode.
2. Choose tool: Arrow, Text, Blur, Crop, Rect, or Ellipse.
3. For crop, press `Enter` or click `Apply Crop`.
4. For Arrow/Rect/Ellipse, choose line style (`Solid`, `Dashed`, `Dotted`), color, and width.
5. For Arrow, choose mode (`Classic`, `Double`, `Line`).
6. Use Undo/Redo (`Ctrl+Z`, `Ctrl+Y`, `Ctrl+Shift+Z`) if needed.
7. Use Copy to write latest version to clipboard.

## 5. Saving Workflow
- `Save PNG` exports lossless output.
- `Save JPG` exports compressed output using options quality.

## 6. Settings
Open extension options page from Chrome extension details.
Available settings:
- Auto-copy on capture
- Capture mode (`Full` / `Clipboard only`)
- Default save format
- JPG quality
