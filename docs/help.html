QuickShot Help

Fast area screenshots with instant clipboard copy and a lightweight editor.

Start Capture

  • Click the QuickShot toolbar icon.
  • Use the shortcut Ctrl+Shift+S.
  • Use the middle mouse button on the page.
  • Use the context menu item Capture area with QuickShot.

Select Area

  • Hold the left mouse button and drag across the page.
  • Release the mouse button to capture.
  • Press Esc to cancel.

Clipboard

  • QuickShot copies the image immediately after capture by default.
  • If copy fails, the editor still opens with the captured screenshot preserved.
  • In the editor, use Copy or Ctrl+C to retry.
  • Use Clipboard only mode in settings to skip opening the editor.

Editor

  • Tools: Arrow, Text, Blur, Crop, Rect, Ellipse.
  • Actions: Undo, Redo, Copy, Save PNG, Save JPG.
  • Drag Out lets you drag the image into another window.
  • Line styles: Solid, Dashed, Dotted.
  • Choose color and width for arrows and shapes.
  • Arrow modes: Classic, Double, Line.
  • Confirm crop with Enter or Apply Crop.
  • Undo and redo shortcuts: Ctrl+Z, Ctrl+Y, Ctrl+Shift+Z.
