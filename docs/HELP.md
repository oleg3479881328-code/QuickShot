# Help

## Start Capture
Use one of the following:
- Left click on toolbar icon (`QuickShot`) for instant start.
- Shortcut: `Ctrl+Shift+S`.
- Middle mouse click on page (site/context dependent).
- Page right-click menu: `Capture area with QuickShot`.

## Capture Interaction
- Drag on page to choose area.
- Release mouse to capture.
- Press `Esc` to cancel selection.

## Clipboard Behavior
- By default, QuickShot copies cropped image immediately.
- If copy fails, editor still opens with screenshot preserved.
- In editor, use `Copy` or `Ctrl+C` to retry copy.
- You can switch to `Clipboard only` mode in Options to skip opening the editor after capture.

## Editor Basics
Tools:
- Arrow
- Text
- Blur
- Crop
- Rect
- Ellipse

Actions:
- `Undo` / `Redo`
- `Drag Out` (click to enable mode, then drag from canvas to another window)
- `Copy`
- `Save PNG`
- `Save JPG`
- `Apply Crop` button

Style controls:
- Line style: `Solid`, `Dashed`, `Dotted`
- Stroke color picker
- Stroke width slider
- Arrow mode: `Classic`, `Double`, `Line`

Crop confirmation:
- Select area with Crop tool.
- Press `Enter` or click `Apply Crop`.
