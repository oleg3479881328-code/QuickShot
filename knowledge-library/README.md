# Central Knowledge Library

CKL — центральная библиотека инженерных решений платформы.

Назначение:

- хранить reusable knowledge
- предотвращать повторное изобретение решений
- давать агентам и разработчикам доступ к проверенным паттернам

Типы знаний:

- architecture patterns
- backend patterns
- frontend patterns
- devops patterns
- ai-agent patterns
- github solutions
- code snippets
- prompt patterns
- runbooks
- templates

Все элементы CKL должны оформляться как Markdown-документы.
